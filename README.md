Project Overview

This project is an interactive data visualization dashboard created using Tableau. It explores Airbnb listing data to uncover trends in pricing, location, and revenue.

The dashboard enables users to quickly understand how different factors such as bedroom count and zip code impact listing prices and overall performance.

 Objectives
Analyze pricing trends based on number of bedrooms
Compare average prices across different zip codes
Visualize revenue trends over time
Understand distribution of listings by bedroom count

 Dashboard Features
 
 Average Price by Bedroom
Bar chart showing the relationship between bedroom count and price
Insight: Prices increase as the number of bedrooms increases

 Distinct Count of Bedrooms
Displays the number of listings per bedroom category
Highlights the most common property sizes

 Price per Zip Code (Map View)
Geographical visualization of pricing distribution
Helps identify high-value and low-value areas

 Revenue Over Time
Line chart tracking revenue trends across 2016
Useful for identifying seasonality and growth

 Price by Zip Code (Detailed)
Comparative bar chart of average prices by zip code
Allows ranking of areas based on pricing
